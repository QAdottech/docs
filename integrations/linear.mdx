## Linear

Linear is a modern project management tool designed for software teams to track issues, manage tasks, and streamline workflows. It offers a clean interface, keyboard shortcuts, and integrations with tools like GitHub and Slack. With features like sprints, roadmaps, and automation, Linear helps teams stay focused and efficient.

## Linear Integration

Connecting QA.tech to your Linear account is simple. Follow these steps.

<Steps>
  <Step title="Connect to Linear">
    Go to [**Settings →
    Connections**](https://app.qa.tech/dashboard/current-project/settings/organization/connections),
    and add the Linear integration. Follow the OAuth flow to grant access and
    set your preferred permissions.
  </Step>
  <Step title="Configure the Project">
    Navigate to [**Settings →
    Integrations**](https://app.qa.tech/dashboard/current-project/settings/integrations?focus=linear).
    Select the Linear Team where you want created issues to appear.
  </Step>
  <Step title="Export an Issue">
    In the Issues section, select an issue and click the Send to Linear button
    to export it.
  </Step>
</Steps>
